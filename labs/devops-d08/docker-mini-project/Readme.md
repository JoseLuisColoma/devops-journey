mini-proyecto DevOps real con Docker: una aplicación Node.js + PostgreSQL + Nginx usando Docker Compose.

- Objetivo:

# Backend → Node.js API

# Base de datos → PostgreSQL

# Proxy → Nginx

- Todo levantado con 1 solo comando

docker-mini-project
│
├── docker-compose.yml
│
├── app
│   ├── server.js
│   └── package.json
│
└── nginx
    └── default.conf
